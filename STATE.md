## 5. 更新规则

- 一次运行只更新与本次动作对应的章节；
- Run ID 必须与报告文件名及 `runs/README.md` 一致；
- 近期论文监控的“已覆盖至”只能写入本次实际检查到的日期；
- 历史搜索记录只描述实际范围，不宣称某个方向已经完整覆盖；
- 已有候选信息更新只记录本次实际检查范围；
- 不在本文件维护论文级状态、下一轮推荐、待跟踪论文或质量判断。
