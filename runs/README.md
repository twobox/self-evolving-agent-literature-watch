# Run Index

本文件索引三种独立动作的运行报告。每次实际运行只追加一行，不在索引中维护推荐、观察、忽略等分类统计。

## 运行报告命名

运行报告路径使用：

```text
runs/YYYY/YYYY-MM-DD-NN.md
```

其中：

- `NN` 是同一天从 `01` 开始的两位序号；
- Run ID 等于不含 `.md` 的文件名，例如 `2026-06-27-01`。

## 更新规则

- 运行类型只使用：`近期论文监控`、`历史优质论文搜索`、`已有候选信息更新`；
- 每次实际运行追加一行，即使新增或更新数量为 0；
- `新增或更新` 必须与对应运行报告一致；
- `报告` 使用指向仓库内 Markdown 文件的相对链接；
- 新增首条记录时删除“当前尚无运行记录”，保留标题和表头；
- Run ID、报告文件名以及 `STATE.md` 中对应动作的记录必须一致；
- 不为同一次运行创建多个索引行。
