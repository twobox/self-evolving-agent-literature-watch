## 2. 本次主要发现

Agent Memory 正从静态文本检索转向学习如何写入、更新、淘汰和利用记忆。MemoPilot 与 Memory-R1 训练记忆管理策略，TMEM 更将经验在线吸收到 LoRA fast weights，使后续策略发生参数级变化。

评测也从单次回忆转向连续任务流中的迁移、遗忘和负迁移。Evo-Memory 提供流式基准，AFTER 测量程序性技能跨任务、跨角色和跨模型迁移；相关诊断工作则揭示错误传播、检索竞争和前向迁移的权衡。

优先阅读建议为已标注 ICML 2026 的 MemoPilot、AdaMEM，以及历史基线 ExpeL、Agent Workflow Memory 和 Reflexion。
