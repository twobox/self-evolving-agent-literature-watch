## 2. 本次主要发现

本轮最明显的变化是，Agent Memory 正从“保存并检索历史文本”转向对完整记忆生命周期进行控制。较早的 ExpeL、AutoGuide 和 Agent Workflow Memory 主要把轨迹压缩为经验、规则或工作流；较新的 Memory-R1、ReMe、ProcMEM 与 MemoPilot 则进一步学习何时写入、更新、删除、验证和复用记忆，使记忆更新本身成为可优化的策略。

第二条主线是程序性记忆和经验压缩。工作流、技能、规则和结构化程序逐渐成为比原始轨迹更高压缩率、更易迁移的经验载体。Agent Workflow Memory、ReMe、ProcMEM 以及 2026 年 6 月新发布的 Managing Procedural Memory 都直接考察了经验能否跨任务、跨角色或跨模型复用。

第三条主线是评测从静态问答转向连续任务流和长期行为。How Memory Management Impacts LLM Agents 揭示了错误传播与错配经验回放问题；Evo-Memory 和 LongMemEval-V2 则把测试时学习、持续记忆更新和环境经验内化作为独立评测目标。当前证据说明“记得更多”并不等于“长期表现更好”，选择性写入、删除、压缩和适用性判断是后续监控的关键维度。

本次近期窗口确认两篇值得优先跟踪的新作：MemoPilot 将多轮记忆更新训练为强化学习问题，但目前主要验证于两个博弈环境；Managing Procedural Memory 提供跨任务、跨角色和跨模型的程序性技能评测，但作为刚公开的预印本，仍需等待更广泛复现和正式发表信息。
