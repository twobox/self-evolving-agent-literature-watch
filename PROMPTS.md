# 可复制的运行提示词

## 近期论文监控

```text
请在当前仓库执行一次近期论文监控。先读取 AGENTS.md，根据标题和摘要判断相关性，记录首次公开或发表状态以及作者机构，并更新候选表和运行记录。
```

指定时间范围：

```text
请在当前仓库监控 [开始日期] 至 [结束日期] 新公开的相关论文。先读取 AGENTS.md。
```

## 历史优质论文搜索

```text
请在当前仓库搜索 [研究方向] 的历史优质论文。先读取 AGENTS.md，根据标题和摘要判断相关性，记录发表和引用情况，并更新候选表和运行记录。
```

## 已有候选信息更新

```text
请在当前仓库更新已有候选论文的信息。先读取 AGENTS.md，核查 candidates.md 中论文的公开时间、发表状态、论文链接、标题和作者机构，并更新运行记录。
```

指定范围：

```text
请更新 candidates.md 中 [论文或序号范围] 的信息。先读取 AGENTS.md。
```

## 方案讨论

```text
请根据当前仓库规则讨论 [运行类型] 的执行方案，不修改仓库文件。
```
